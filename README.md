# Online Course Recommendation System

An end-to-end course recommendation system that combines deep learning–based personalized recommendations, smart course search, and new-user onboarding.

## Project Overview

This project recommends online courses using multiple recommendation techniques and selects the best-performing deep learning model for final personalized recommendations.

The system supports:
- personalized recommendations for existing users
- smart course search using keyword-based retrieval
- cold-start onboarding for new users
- exact course selection and related course recommendations
- deployed frontend for interactive usage

## Features

### 1. Existing User Recommendation
Existing users can enter their user ID and get:
- their previous course history
- personalized course recommendations generated by the final deep learning model

### 2. Smart Course Search
Users can search using keywords such as:
- python
- machine
- ai
- cyber
- photo

The system handles:
- partial words
- abbreviations
- similar terms
- fuzzy-like matching via alias normalization and retrieval logic

### 3. New User Onboarding
For new users without prior history:
- search for a course
- review matched courses with instructor, difficulty, and rating
- select one exact course
- create a new user profile
- get related recommendations based on that selected course

### 4. Stored New Users
New users are stored with:
- user ID
- course name
- instructor
- difficulty level
- rating

This helps simulate a practical cold-start recommendation flow.

## Recommendation Models Implemented

The following recommendation approaches were built and tested:

### Traditional Recommendation Systems
- Popularity-Based Recommendation
- Content-Based Recommendation
- Collaborative Filtering
- Hybrid Recommendation

### Deep Learning Models
- Basic Embedding Model
- Enhanced Embedding Model
- Wide & Deep Model
- DeepFM Model

## Final Best Model

The final selected model is:

**Enhanced Embedding Recommender**

It performed best in the final comparison based on:
- lowest test loss
- lowest test MAE

This model is used for personalized recommendations for existing users.

---

## Search and Retrieval Layer

A course knowledge base was created at the course level, and a TF-IDF retriever was built on top of it.

This enables:
- query-based course search
- smart course matching
- retrieval for cold-start new users
